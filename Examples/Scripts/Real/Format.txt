0. Токен - базовый элемент грамматики языка.
1. Все "токены" разделяются пробелами. Пример:
 A B C - три токена - A B C
 A B + - три токена - A B +
 A B+ - два токена - A B+
 A+ B - два токена - A+ B
 A+B - один токен - A+B
2. Токен // - определяет строковый комментарий. Парсер должен пропустить все символы от // до конца строки (#13#10).
3. Токен /* - определяет блочный комментарий. Парсер должен пропустить все символы от /* до */.
